# Screenshots

Two screenshots are referenced by `ldapreauth.xml` and are required before
submitting to the GLPI plugins catalog. Add them here with these exact names:

- `screenshot-config.png` — the config tab under **Setup > General > LDAP Re-auth on Approval**.
- `screenshot-approval.png` — a ticket timeline approval showing the injected
  **Windows username / Windows password** fields below the comment box.

PNG, reasonably sized (e.g. ~1200px wide). These files are excluded from the
plugin release archive via `.gitattributes` / the release workflow.
